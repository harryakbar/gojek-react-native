# Gojek App Homescreen Clone

A React Native clone of the [Gojek](https://www.gojek.com/) app homescreen, built as a UI exploration project.

## Tech Stack

- **Framework:** React Native
- **Language:** TypeScript
- **Package Manager:** Yarn

## Getting Started

### Prerequisites

- Node.js >= 14
- Yarn
- React Native CLI
- Xcode (for iOS) or Android Studio (for Android)

### Install

```bash
yarn install
```

### Run

**iOS:**

```bash
yarn ios
```

**Android:**

```bash
yarn android
```

## Notes

This project was built for learning and UI exploration purposes — recreating the Gojek homescreen layout using React Native components.
